Methodology

The IoT and AI-based smart plant watering system ensures efficient irrigation and water conservation using sensor-driven automation and machine learning.

  • ESP32 Microcontroller: Serves as the central processing unit, integrating data from multiple sensors and controlling actuators for efficient irrigation.
  • Sensors:
    • Soil moisture sensors detect real-time moisture levels and relay data to the microcontroller.
    • DHT11 records temperature and humidity, which are crucial for plant health and irrigation adjustments.
    • Water flow sensors track water usage to monitor efficiency and detect anomalies.
  • Watering System:
    • Solenoid valves regulate water flow based on real-time sensor readings.
    • A relay module controls valve activation, ensuring precise irrigation.
    • A water pump maintains a steady supply, optimizing water distribution.
  • Modes of Operation:
    • Manual Mode: Users control watering via a mobile app with real-time monitoring.
    • Auto Mode: The system autonomously waters plants based on soil moisture levels, reducing human intervention.
    • AI Mode: Utilizes machine learning to predict water requirements, adjusting based on environmental parameters.
  • Model Training:
    • Sensor data (temperature, humidity, soil moisture, and water flow) is collected and preprocessed.
    • Two linear regression models are trained:
      • User Model: Trained on user watering patterns to replicate user behavior.
      • Optimal Model: Trained using standard plant growth requirements to determine ideal water usage.
    • Google Colab is used for training, leveraging Python libraries such as NumPy, Pandas, and Scikit-learn.
    • The trained models are deployed on the ESP32 microcontroller, enabling real-time irrigation adjustments.
    • The AI models continuously update and improve over time with new data, refining their predictions for better efficiency.
    • The system compares user watering habits with optimal watering recommendations to improve conservation and efficiency.
  • System Integration:
    • The microcontroller communicates with cloud storage to log data and improve machine learning insights.
    • Users can access system updates and analytics via our website.
    • Wireless communication ensures seamless remote monitoring and control.
  • Dataset:
    • We are using a publicly available dataset from Kaggle for predicting plant watering needs: Dataset for Predicting Watering the Plants.
    • The dataset contains temperature, humidity, soil moisture, which help train and refine the AI models.
